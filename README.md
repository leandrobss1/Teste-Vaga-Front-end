# Econverse Store

Econverse Store é um projeto front-end desenvolvido como solução de um desafio técnico. O objetivo foi construir uma interface moderna de e-commerce com foco em componentização, responsividade, organização de código e boas práticas de desenvolvimento.

# Tecnologias utilizadas

- React
- TypeScript
- Vite
- Styled-components
- SCSS (Sass)
- SwiperJS
- React Icons

# Estilização

O projeto utiliza uma abordagem híbrida de estilização:

- Styled-components: para componentes dinâmicos e encapsulados
- SCSS (Sass): para estilos isolados organização estrutural do CSS

Essa abordagem foi escolhida para melhorar a escalabilidade, organização e manutenção do projeto.

# Como executar o projeto

# 1. Clone o repositório

git clone https://github.com/seu-usuario/econverse-store.git

# 2. Acesse a pasta do projeto

cd econverse-store

# 3. Instale as dependências

npm install

# 4. Execute o projeto

npm run dev

# Funcionalidades

- Listagem de produtos
- Carrossel de produtos com SwiperJS
- Modal de detalhes do produto
- Controle de quantidade de itens
- Newsletter interativa com validação visual
- Footer institucional com links organizados
- Layout moderno e responsivo
